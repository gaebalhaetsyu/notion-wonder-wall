# notion-wonder-wall
할 일을 완료하며 벽돌을 하나씩 허물고, 벽 너머의 숨겨진 풍경을 발견하는 투두(To-Do) 위젯
## ✨ 주요 기능 (필수 토핑)
- **벽돌 해체 진행도**: 투두 완료율에 따라 벽돌이 하나씩 제거되며 진행률을 시각적으로 확인합니다.
- **숨겨진 장면 공개**: 모든 벽돌이 제거되면 전체 풍경과 함께 무작위 응원/보상/T식 팩폭 메시지가 나타납니다.

## 🛠️ 기술 스택 & 데이터
- **배포**: Vercel
- **아이콘**: Lucide Icons
- **폰트**: Google Fonts (Noto Sans KR 등)
- **데이터**: Unsplash API (풍경 이미지), 자체 제작 메시지 리스트(JSON)
